Catalogues d'images du wiki (héros et ressources), détectés automatiquement depuis le dépôt, et matrice du tableau comparatif. Le regroupement des images en bandes se fait carte par carte, dans chaque onglet de type « Héros ».
Catalogue des héros
Les images sont détectées automatiquement dans le dossier images/heros/ du dépôt (jpg, png, webp…). Déposez-y vos fichiers, puis donnez un nom à chaque héros (utilisé dans le tableau comparatif et au survol). Pour les afficher, cliquez-les directement dans une carte d'un onglet « Héros ».
Catalogue des ressources
Les images sont détectées automatiquement dans le dossier images/ressources/ du dépôt (jpg, png, webp…). Il suffit d'y déposer vos fichiers : aucun ajout manuel ici. Pour les afficher, cliquez-les directement dans une carte d'un onglet « Héros ».
Tableau comparatif
Cochez les capacités de chaque héros pour le tableau "Comparatif rapide". Ce tableau est global ; chaque onglet Héros choisit de l'afficher ou non.

Miroir seed-data.json
Le fichier seed-data.json du dépôt sert au premier affichage du site (avant Firestore). Après chaque enregistrement, téléchargez-le ici puis remplacez-le dans le dépôt (git push) pour qu'il reste fidèle au contenu en ligne — sinon les liens partagés vers les nouveaux onglets et le premier rendu seront périmés.
